# Student Record Management System (Java)

Student Record Management System is a console-based Java application developed to manage student information efficiently. It allows users to perform CRUD operations such as adding, viewing, updating, and deleting student records. The project is implemented using Object-Oriented Programming principles and Java Collections like ArrayList and HashMap, with proper input validation and basic exception handling.

## Technologies Used
1.Java
2. OOP Concepts
3. Collections Framework

## Features
1.Add student
2. View students
3. Update student details
4. Delete student records
